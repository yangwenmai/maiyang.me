## time.ParseDuration

支持 `-24.5h` / `2h3m4s` 等格式，非常的灵活。

```golang
pd, err := time.ParseDuration("2h3m4s")
if err != nil {
	panic(err)
}
time.Now().Add(pd)
```

## time.Add

```golang
time.Now().Add(-24*time.Hour)
```

## time.Sub

```golang
time.Now().Sub(startTime)
```
